# 查理·芒格思维模型认知卡片

一个交互式的思维模型学习工具，帮助你更好地理解和运用查理·芒格的思维模型。

## 项目简介

本项目是一个基于网页的思维模型学习工具，收录了查理·芒格推崇的100多个思维模型。通过交互式卡片的形式，帮助用户更直观地学习和理解这些模型，从而提升思维能力和决策水平。

## 功能特点

- 🎯 **模型分类**：将思维模型按照不同学科领域（心理学、数学、物理学、经济学等）进行分类，方便查找和学习
- 🔄 **卡片翻转**：通过翻转卡片的交互方式，正面显示模型名称和简介，背面展示详细解释和应用场景
- 🔍 **智能搜索**：支持模型名称和内容的快速搜索，轻松找到感兴趣的思维模型
- 📌 **收藏功能**：可以收藏常用或重要的思维模型，方便日后复习和参考
- 📱 **响应式设计**：完美适配桌面端和移动端，随时随地学习思维模型

## 使用说明

1. **浏览模型**
   - 打开网页后，可以看到所有思维模型的卡片展示
   - 点击卡片可以查看模型的详细说明
   - 使用页面顶部的分类标签筛选不同类别的模型

2. **搜索功能**
   - 在顶部搜索框输入关键词
   - 系统会实时显示匹配的思维模型

3. **收藏管理**
   - 点击卡片右上角的书签图标可以收藏/取消收藏
   - 点击顶部导航栏的"收藏"按钮查看已收藏的模型

## 本地部署

### 环境要求

- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- Node.js（可选，如果需要启动本地服务器）

### 部署步骤

1. 克隆项目到本地：
   ```bash
   git clone [项目地址]
   cd ThinkingModel
   ```

2. 方式一：直接打开
   - 直接用浏览器打开 `index.html` 文件即可使用

3. 方式二：启动本地服务器（推荐）
   - 安装 Node.js
   - 安装简单的静态文件服务器：
     ```bash
     npm install -g http-server
     ```
   - 在项目根目录运行：
     ```bash
     http-server
     ```
   - 打开浏览器访问 `http://localhost:8080`
